# Linear Regression

Build a simple linear regression model to predict continuous values. Learn the basics of supervised learning with Deepbox.

## Deepbox Modules Used

| Module               | Features Used     |
| -------------------- | ----------------- |
| `deepbox/ml`         | LinearRegression  |
| `deepbox/ndarray`    | tensor            |
| `deepbox/metrics`    | r2Score, mse, mae |
| `deepbox/preprocess` | trainTestSplit    |

## Usage
