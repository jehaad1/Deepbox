# Logistic Regression

> **View online:** https://deepbox.dev/examples/08-logistic-regression

Build a binary classification model using logistic regression. Learn to classify data into two categories using the Iris dataset.

## Deepbox Modules Used

| Module               | Features Used                                         |
| -------------------- | ----------------------------------------------------- |
| `deepbox/datasets`   | loadIris                                              |
| `deepbox/ml`         | LogisticRegression                                    |
| `deepbox/metrics`    | accuracy, precision, recall, f1Score, confusionMatrix |
| `deepbox/preprocess` | trainTestSplit, StandardScaler                        |

## Usage

```bash
npm run example:08
```

## Output

- Console output showing classification metrics and confusion matrix
