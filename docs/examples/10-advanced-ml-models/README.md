# Advanced ML Models

> **View online:** https://deepbox.dev/examples/10-advanced-ml-models

Demonstrates advanced ML models: KMeans clustering, K-Nearest Neighbors (classification and regression), PCA dimensionality reduction, and Gaussian Naive Bayes.

## Deepbox Modules Used

| Module               | Features Used                                                      |
| -------------------- | ------------------------------------------------------------------ |
| `deepbox/ml`         | KMeans, KNeighborsClassifier, KNeighborsRegressor, PCA, GaussianNB |
| `deepbox/ndarray`    | tensor                                                             |
| `deepbox/metrics`    | accuracy                                                           |
| `deepbox/preprocess` | trainTestSplit                                                     |

## Usage

```bash
npm run example:10
```

## Output

- Console output showing clustering labels, KNN accuracy, PCA variance, and Naive Bayes predictions
