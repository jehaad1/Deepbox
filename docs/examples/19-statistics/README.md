# Statistical Analysis

> **View online:** https://deepbox.dev/examples/19-statistics

Perform descriptive statistics and hypothesis testing. Analyze data distributions and test hypotheses.

## Deepbox Modules Used

| Module            | Features Used                                                                   |
| ----------------- | ------------------------------------------------------------------------------- |
| `deepbox/ndarray` | tensor                                                                          |
| `deepbox/stats`   | mean, median, std, variance, skewness, kurtosis, percentile, pearsonr, corrcoef |

## Usage

```bash
npm run example:19
```

## Output

- Console output showing descriptive statistics, percentiles, Pearson correlation, and correlation matrix
