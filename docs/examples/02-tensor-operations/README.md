# Tensor Operations

> **View online:** https://deepbox.dev/examples/02-tensor-operations

Explore arithmetic, mathematical, and reduction operations on tensors. Deepbox supports 90+ tensor operations with full broadcasting.

## Deepbox Modules Used

| Module            | Features Used                                                     |
| ----------------- | ----------------------------------------------------------------- |
| `deepbox/ndarray` | add, sub, mul, div, sqrt, exp, log, sin, cos, sum, mean, max, min |

## Usage

```bash
npm run example:02
```

## Output

- Console output demonstrating arithmetic, math functions, trigonometry, and axis-wise reductions
