# Linear Algebra Operations

Explore matrix decompositions and linear system solving. Essential for understanding ML algorithms under the hood.

## Deepbox Modules Used

| Module            | Features Used                             |
| ----------------- | ----------------------------------------- |
| `deepbox/linalg`  | det, inv, trace, norm, svd, qr, lu, solve |
| `deepbox/ndarray` | tensor                                    |

## Usage

```bash
npm run example:20
```

## Output

- Console output showing determinant, trace, inverse, norms, SVD/QR/LU decompositions, and linear system solutions
