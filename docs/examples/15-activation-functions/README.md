# Activation Functions

Explore different activation functions used in neural networks. Learn when to use each activation function.

## Deepbox Modules Used

| Module            | Features Used                                                                 |
| ----------------- | ----------------------------------------------------------------------------- |
| `deepbox/ndarray` | relu, sigmoid, softmax, gelu, leakyRelu, elu, mish, swish, softplus, linspace |
| `deepbox/plot`    | Figure, plot                                                                  |

## Usage

```bash
npm run example:15
```
